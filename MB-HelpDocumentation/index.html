System Requirements

  • Monitor Resolution has to be 1024 x 768
  • 2 Core Processor
  • Requires Java
  • Operating System: Windows
  • Files have to be formatted in .CSV

    • *If recent company changes have changed Bun Manifest formatting, naming, file formats. Pallet Manifest Generator may not be able to convert files properly

    Importing Bun Manifest

  • Browse for the bun manisfest you are looking for and select it
  • A preview should show you the generated manifest for the file.
  • You can choose wether to print or to export to a word file.
  • One done you can choose another bun manifest or just close the program.

    • *Important: the bun manifest has to be in a .CSV file format not Excel or any other.
